# Casa Ceci - Sitio Web

Pequeño sitio web para un alojamiento de alquiler temporal, implementado con HTML5, CSS3 y JavaScript puro.

Estructura:
- index.html - Inicio
- alojamiento.html - Detalles y galería
- reservas.html - Formulario de reservas (validaciones JS)
- ubicacion.html - Dirección y mapa embebido
- contacto.html - Formulario de contacto (validaciones JS)
- css/styles.css - Estilos compartidos
- js/script.js - Validaciones e interacciones
- assets/ - Imágenes y recursos

Instrucciones:
- Abrir `index.html` en un servidor web (recomendado) o con Live Server en VS Code.
- Reemplazar las imágenes de ejemplo en `assets/` si lo desea.
